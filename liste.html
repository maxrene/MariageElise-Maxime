Votre présence à notre mariage est le plus beau des cadeaux que vous puissiez nous faire. Cependant, si vous souhaitez nous témoigner une attention supplémentaire, vous trouverez sur cette page quelques idées qui nous feraient plaisir.

Habitant à l'étranger, nous avons opté pour une liste dématérialisée. Pour éviter les frais, deux options s'offrent à vous : un virement via l'application Revolut ou, pour les cadeaux spécifiques, la possibilité de l'acheter vous-même. Dans tous les cas, n'oubliez pas de cliquer sur le bouton de confirmation sur le site pour que votre participation soit bien enregistrée. Merci du fond du cœur pour votre générosité !
